ActionController mock object for testing.

This provides just the basics we need for testing our other gems. You probably won’t use this class.

Example

  require 'action_controller_mock'
  class ApplicationController < ActionController::Base
    before_filter :foo  # mock
    after_filter :bar   # mock
  end
